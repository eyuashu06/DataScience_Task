# Titanic Data Analysis Project

## Overview
This project analyzes the Titanic dataset using pandas.

## Steps Performed
- Data Exploration
- Data Cleaning
- Data Analysis (groupby)
- Filtering
- Insights

## Key Insights
- Females had higher survival rates
- 1st class passengers survived more
- Children were prioritized

## Tools Used
- Python
- Pandas
- Google Colab
