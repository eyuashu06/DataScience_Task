# DataScience_Task

## 📌 Overview
This repository contains three levels of Python mini-projects ,the first task in DataScience Each level increases in complexity, from basic logic to an interactive CLI game.

---

## 🟢 Level 1: Prime Number Finder

### 🔹 Description
A Python program that checks whether a given number is a **prime number**.

### 🔹 Features
- Accepts user input
- Determines if a number is prime
- Displays the result clearly

### 🔹 Concepts Used
- Conditional statements (`if-else`)
- Loops (`for`)
- Basic mathematics



---

## 🟡 Level 2: Fibonacci & Vowel Counter

### 🔸 Fibonacci Number Generator

#### 🔹 Description
Generates Fibonacci sequence up to a specified number of terms.

#### 🔹 Features
- Displays Fibonacci sequence
- User-defined number of terms


---

### 🔸 Vowel Counter

#### 🔹 Description
Counts the number of vowels in a given string.

#### 🔹 Features
- Case-insensitive vowel counting
- Works with any text input



---

## 🔴 Level 3: Tic-Tac-Toe (CLI Game)

### 🔹 Description
A command-line based Tic-Tac-Toe game for two players.

### 🔹 Features
- Two-player gameplay (X and O)
- CLI board display
- Win and draw detection
- Input validation

### 🔹 How to Play
1. The board has positions from 1 to 9.
2. Players take turns choosing positions.
3. Player 1 uses **X**, Player 2 uses **O**.
4. The game ends when:
   - A player wins, or
   - The board is full (draw)
